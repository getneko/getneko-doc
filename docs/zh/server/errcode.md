# 错误码

当code为0时正常，为负数时错误。  

| 错误代码 | 原因                     |
|---------|--------------------------|
| -1      | 请求参数不合法           |
| -2      | 用户名或邮箱已经存在     |
| -3      | 用户名不存在或者密码错误 |
| -4      | 用户没有登录             |
| -5      | 项目不存在或者你不是创建者|
| -6      | 项目已经存在             |
| -7      | 项目不存在               |
| -8      | 用户没有权限             |
| -9      | 用户不存在               |
| -10     | 项目必须有管理员         |
| -11     | 创建者必须为管理员       |
| -12     | 账户中还没有项目         |
| -13     | api不存在                |
